## 🆘 ЕСЛИ ПОЯВИЛИСЬ ПРОБЛЕМЫ

### Сервер не запускается
```bash
# 1. Убедитесь что PostgreSQL работает
pg_isready -h localhost -p 5432

# 2. Проверьте БД существует
psql -U postgres -h localhost -l

# 3. Выполните миграции заново
python manage.py migrate
```

### Ошибка при подключении к БД
```bash
# Создайте БД если не существует
psql -U postgres -h localhost -c "CREATE DATABASE shop;"

# Выполните миграции
python manage.py migrate
```

### Забыли пароль администратора
```bash
# Переустановите
python manage.py changepassword admin
```
